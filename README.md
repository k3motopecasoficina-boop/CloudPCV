# ☁️ CloudVM

## 📱 Sobre
App de Máquinas Virtuais REAIS no celular

## 🖥️ Funcionalidades
- Criar VM com Windows/Linux/Ubuntu
- Definir RAM e Armazenamento
- Rodar SO REAL com QEMU
- Apps ORIGINAIS (Chrome, Spotify, etc)

## 🚀 Como usar
1. Abra o app
2. Clique em "Criar VM"
3. Escolha as configurações
4. Pronto! VM rodando!
